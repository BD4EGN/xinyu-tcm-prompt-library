# GPT Image 2 Prompt 工程指南（中医养生专用）

## 核心认知：GPT Image 2 不是关键词引擎

GPT Image 2 是带推理层的原生多模态模型。
这意味着：不要像写 Midjourney 那样堆关键词，要用自然语言描述画面。

## 分层法（Layer Method）

这是社区验证最有效的方法，也是本库所有模板的底层结构。

```
第 1 层：主体（Subject）
  描述主体是谁、长什么样、周围有什么
  例："一个陶瓷茶杯放在竹制茶盘上，旁边有几朵干菊花，
      杯子冒着热气"

第 2 层：场景（Setting）
  在哪里？什么环境？
  例："阳光透过竹帘洒在木桌上，背景是虚化的中式书房，
      有绿色盆栽"

第 3 层：风格（Style）
  什么视觉风格？
  例："新中式/新国潮风格，柔和温暖色调，小红书封面风格"

第 4 层：构图（Composition）
  怎么拍？什么角度？
  例："竖屏 3:4，主体居中偏下，上方留白 40% 放文字"

第 5 层：用光（Lighting）
  光线从哪来？什么氛围？
  例："柔和晨光从左侧斜射，温暖金色调，带朦胧光晕"

第 6 层：文字（Typography）
  精确的文字排版
  例："标题：'XXXX' 白色粗体，占画面宽度 60%，
      副标题：'XXXXXXXX' 浅灰色小字。无其他文字。"
```

## 5 个必须掌握的核心技巧

### 技巧 1：精确文字必须加引号

```
❌ 弱：写个标题'湿气重的信号'
✅ 强：图片上方中央放置标题文字："湿气重的5个信号"
       白色粗体，字号占画面宽度 60%，带阴影增强可读性
```

遇到非常用英文词，逐个字母拼写：
```
Display the brand name "ZEITGEIST" (Z-E-I-T-G-E-I-S-T)
```

### 技巧 2：必加反文字防护

GPT Image 2 太爱乱加文字了，不加一定有随机英文出现：

```
图片中除了指定的标题文字外，没有任何其他文字、标签、
水印或随机字母。
```

实测：不加约 60% 图有额外文字，加上后降到 10% 以下。

### 技巧 3：6 元素按优先级排序

顺序即优先级：

```
1. 主体  →  2. 场景  →  3. 构图  →  4. 用光  →  5. 风格  →  6. 文字
```

越靠前的元素越重要。冲突时模型优先保证前面的。

### 技巧 4：迭代不超过 3 次

同一对话里改 3 次以上，阴影和光照开始产生噪点退化。
超过 3 次改不满意 → 新建对话重写 prompt。

### 技巧 5：使用参考图锁定品牌风格

GPT Image 2 支持最多 16 张参考图。

```
上传 3 张图：
- 图 1：你的色板参考（色调）
- 图 2：你的排版参考（布局）
- 图 3：爆款封面（构成）
Prompt："参考图 1 的色调，图 2 的排版，
         图 3 的构成方式，生成新的封面。"
```

## 常见图片类型的最佳 Prompt 长度

| 图片类型 | 最佳 Prompt 字数 | 推荐模式 |
|---------|----------------|---------|
| 简单封面（单主体+标题） | 50-100 字 | 即时模式（3-5秒） |
| 图文卡片（多元素+正文） | 100-200 字 | 思考模式（10-15秒） |
| 复杂场景（多人+多物+文字） | 200-400 字 | 思考模式（15-30秒） |

## 中医养生的语言策略

### 描述视觉元素的词汇

| 中文 | 英文 prompt 关键词 |
|------|------------------|
| 温润 | warm, smooth, subtle glow |
| 古朴 | rustic, vintage, time-worn |
| 宁静 | serene, tranquil, peaceful |
| 自然 | natural, organic, earthy |
| 雅致 | elegant, refined, tasteful |

### 道具关键词

- 陶瓷茶杯 / 紫砂壶
- 竹编 / 木器 / 亚麻
- 中药材（枸杞/菊花/陈皮/黄芪/当归/灵芝）
- 线装书 / 书法 / 国画
- 砚台 / 古琴（aesthetic 用途）
- 盆景 / 竹帘
