# 祛湿系列 · 3 集短视频方案

**总时长：28s + 28s + 30s = 86 秒（90 秒以内）**

---

## 第 1 集：湿气的坏处（28 秒）

| 时间 | 画面 | 口播 |
|------|------|------|
| 0-2s | 深色背景弹出"你有这些情况吗？" | （无口播，文字引导） |
| 2-7s | 症状列表逐条渐入 | 你是不是觉得身上像裹了湿毛巾，沉得不想动？没精神、睡不醒，睡多久都累？体重没怎么吃却在涨，大便粘马桶冲不干净，舌苔厚腻嘴里不清爽？ |
| 7-8s | 金色大字弹出"这些都是湿气重的表现" | 这些都是湿气重。 |
| 10-18s | 切换暖色调画面，三个"湿气→疾病"箭头 | 湿气不除，拖久了问题更大。停在中焦→腹部肥胖代谢差，上蒙清窍→头昏脑涨面色晦暗，下注关节→关节沉重酸痛。时间久了，高血脂、脂肪肝、高尿酸都跟着来。 |
| 20-24s | 全黑背景，中间出现金色光晕 | 那你知道为什么——祛湿这么难吗？ |
| 25-28s | "关注我，看下一条 👉" | 关注我，看下一条。 |

## 第 2 集：为什么祛湿难（28 秒）

| 时间 | 画面 | 口播 |
|------|------|------|
| 0-2s | "祛湿为什么这么难？" 标题渐入 | 祛湿为什么这么难？ |
| 2-9s | 金色文字逐行出现"湿性粘滞、缠绵不去" | 湿性粘滞，缠绵不去。它不像风寒发个汗就走了。湿气像油和面和在一起，粘住了。 |
| 9-10s | 画面切换 | 更麻烦的是——你的生活习惯一直在生湿。 |
| 10-19s | 三个图标（🌙 🧊 ❄️）依次弹出，各配文字 | 熬夜伤脾阳，运化失常，湿从中生。冷饮冰食，寒伤脾胃，水湿不化，湿聚成痰。空调久吹，毛孔闭塞，湿无出路，内陷脏腑。这边在祛，那边在生，没完没了。 |
| 19-24s | 暗色背景文字"反反复复没完没了"→"所以要顺应天时" | 所以祛湿要顺应天时，不是瞎祛。 |
| 24-28s | "下一条告诉你今年为什么更不一样 👉" | 下一条告诉你，今年为什么更不一样。 |

## 第 3 集：丙午年更要祛湿（30 秒）

| 时间 | 画面 | 口播 |
|------|------|------|
| 0-3s | 蓝色冷色调，水面涟漪效果，"丙午年·2026" 徽标 | （无口播，视觉引导） |
| 3-8s | 大字弹出"今年对湿气重的人是最难过的一年" | 今年对湿气重的人来说，是最难过的一年。 |
| 8-10s | 画面过渡 | 五运六气告诉我们—— |

| 15-17s | 底部引用文字渐入 | 《脉诀汇辨》说"运克天气，是为不和"。 |
| 19-24s | 深色背景大字"今年祛湿不是可做可不做的事，而是重要的养生手段" | 所以今年祛湿，不是可做可不做的事，而是重要的养生手段。 |

## 项目结构

```
hyperframes-demo/
├── v1-harm/         ← 第1集：湿气的坏处（28s）
│   └── index.html
├── v2-why-hard/    ← 第2集：为什么祛湿难（28s）
│   └── index.html
├── v3-bingwu/      ← 第3集：丙午年更要祛湿（30s）
│   └── index.html
└── script-and-spec.md  ← 本文件
```

## 渲染命令

```bash
# 第1集
cd hyperframes-demo/v1-harm && npx hyperframes render --output v1-harm.mp4

# 第2集
cd hyperframes-demo/v2-why-hard && npx hyperframes render --output v2-why-hard.mp4

# 第3集
cd hyperframes-demo/v3-bingwu && npx hyperframes render --output v3-bingwu.mp4

# 如果本机渲染有 Chrome 超时问题，加 --docker 参数：
npx hyperframes render --docker --output v1-harm.mp4
```
