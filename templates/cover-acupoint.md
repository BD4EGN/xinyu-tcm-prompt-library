# 穴位科普卡 — 模板

## 适用场景
穴位位置介绍、按揉方法、功效说明。

## 分层 Prompt

```
[主体] 简约的人体图示或局部特写，
       用红色圆点标注穴位位置，
       搭配手部按揉的示意姿势

[场景] 浅色纯色背景，干净专业，
       标注线和穴位名称清晰可见

[风格] 医学插画风格，干净清晰，
       现代中医科普风

[构图] 竖屏 3:4，人体/局部在中央偏左，
       右侧 1/3 留白放文字

[用光] 均匀平坦无阴影，医学图示风格

[文字] 左上角标注穴位名："【穴位名】"
       右侧列出功效：
       "功效：
       · 缓解【症状1】
       · 改善【症状2】
       · 调理【症状3】"
       底部小字："每天按揉3分钟"
       无其他文字
```

## 应用示例

| 穴位 | 标题 | 功效 |
|------|------|------|
| 足三里 | "足三里——养生第一大穴" | 健脾和胃、补中益气 |
| 涌泉穴 | "睡前按涌泉，补肾又安神" | 补肾固精、改善睡眠 |
| 三阴交 | "女性的不老穴位" | 调经活血、美容养颜 |
| 合谷穴 | "头痛牙痛按这里" | 镇静止痛、通经活络 |
