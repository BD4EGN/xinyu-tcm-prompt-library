# 知识科普卡 — 通用模板

## 适用场景
每日知识科普类笔记。所有中医养生选题的通用模板。

## 分层 Prompt

```
[主体] 一个精美的陶瓷茶杯放在竹制茶盘上，
       旁边放着几朵干菊花和几粒枸杞，
       杯子冒着缥缈的热气

[场景] 阳光透过竹帘洒在木桌上，
       背景是虚化的中式书房，有书法卷轴和绿色盆栽，
       营造宁静治愈的养生氛围

[风格] 新中式极简风格，小红书封面图，
       暖米色 + 淡茶色 + 自然绿，柔和温馨

[构图] 竖屏 3:4（1080x1440），
       主体居中偏下，上方留白 40% 放置标题文字，
       浅景深

[用光] 柔和的晨光从左侧斜射，温暖金色调，
       带有轻微光晕和朦胧感

[文字] 图片上方中央放置标题文字：
       "【标题】"
       白色粗体大字，带深色投影增强可读性，
       字号占画面宽度 60%，
       标题下方空一行，用小号浅灰色副标题：
       "【副标题】"
       图片中除了上述文字外，没有任何其他文字或标签
```

## 替换指南

| 占位符 | 替换内容 | 示例 |
|--------|---------|------|
| 【标题】 | 核心痛点/反常识/数字 | "湿气重的5个信号" |
| 【副标题】 | 补充说明 | "看看你中了几个" |
| 茶杯+菊花+枸杞 | 根据主题换药材 | 陈皮+茯苓 / 黄芪+当归 |
| 色调 | 根据季节调 | 春绿 / 秋金 / 冬暖棕 |

## 应用示例

### 示例 1：祛湿主题

```
标题："体内湿气重的5个信号"
副标题："中医教你自查"
药材道具：陈皮 + 茯苓 + 薏米
色调：暖米色 + 自然绿
```

### 示例 2：失眠主题

```
标题："长期失眠的3个根源"
副标题："别再只知道喝牛奶了"
药材道具：酸枣仁 + 百合 + 莲子
色调：暖紫色 + 深蓝
```

### 示例 3：养肝主题

```
标题："养肝就是养命"
副标题："熬夜党必看"
药材道具：菊花 + 决明子 + 枸杞
色调：淡绿 + 暖黄
```
