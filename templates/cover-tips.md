# 养生小贴士卡 — 模板

## 适用场景
日常养生习惯分享、便宜好用的养生方法。

## 分层 Prompt

```
[主体] 一只手正在冲泡一杯养生茶/按揉某个穴位，
       （只拍手不拍脸）
       或者一个极简的养生用品场景

[场景] 家中一角/办公桌上，自然生活场景，
       不刻意但干净整洁

[风格] 真实生活感，手机实拍质感（GP Image 2 模拟），
       温馨日常

[构图] 竖屏 3:4，特写或中景，
       主体在画面中心偏右，左侧留白

[用光] 家居暖光，温馨柔和

[文字] 左上角小标签："养生小贴士" 椭圆标签
       画面中部大字："【核心建议】"
       底部小字："每天坚持3分钟" 或 "收藏⭐"
       无其他文字
```

## 应用示例

| 主题 | 标题 | 视觉主体 |
|------|------|---------|
| 晨起一杯水 | "早起第一杯水怎么喝" | 玻璃杯+温水+柠檬片 |
| 泡脚 | "泡脚加一物，效果翻倍" | 木桶+热水+艾草包 |
| 按穴位 | "睡前按这里，一觉到天亮" | 手指按揉脚底/手腕 |
| 呼吸法 | "3分钟腹式呼吸法" | 坐着放松的侧影 |
