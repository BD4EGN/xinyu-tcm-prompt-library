# 问答互动卡 — 模板

## 适用场景
读者问答、误区澄清、互动讨论。

## 分层 Prompt

```
[主体] 简约的对话气泡风格布局，
       或一个问号图形作为主视觉

[场景] 纯色渐变背景（根据话题选择暖色或冷色），
       无实景

[风格] 简洁扁平设计风，小红书问答卡片风格

[构图] 竖屏 3:4，中央大问号图形，
       上半部分放问题，下半部分放答案

[用光] 均匀光，无需特殊光效

[文字] 上半部分："Q：【问题】" 大号字体
       中间：分隔线
       下半部分："A：【答案】" 中号字体
       底部："👇 你有这个问题吗？评论区告诉我"
       无其他文字
```

## 应用示例

```
Q：湿气重能不能喝牛奶？
A：脾虚湿盛的人建议少喝。牛奶性偏寒湿，
   容易加重脾胃负担。可以换成温热豆浆或姜茶。
```
