# 对比测评卡 — 模板

## 适用场景
适合 vs 不适合、正确做法 vs 错误做法、A vs B 对比。

## 分层 Prompt

```
[主体] 左右分屏布局：
       左半部分——柔和绿色渐变背景
       右半部分——柔和灰色渐变背景
       中间用自然的渐变或细线分隔

[场景] 干净的纯色基底，无复杂背景元素

[风格] 极简信息图风格，清晰可读，
       小红书卡片风

[构图] 竖屏 3:4，左右 1:1 分割，
       上下两端各留 15% 空间

[用光] 均匀柔和的光线，无明显方向感

[文字] 顶部中央大标题："【对比题目】"
       左侧上方小标："✅ 适合 / 正确做法" 绿色
       右侧上方小标："❌ 不适合 / 错误做法" 灰色
       左右各列出 2-3 个要点（小号文字）
       底部留白："收藏⭐ 对比不迷路" 浅灰色
       无其他文字
```

## 应用示例

### 示例 1：体质对比

```
标题："你适合祛湿茶吗？"
✅ 适合：舌苔厚腻、大便粘腻、身体困重、面部油腻
❌ 不适合：口干舌燥、手心发热、大便干结、舌头红少苔
```

### 示例 2：冲泡方法对比

```
标题："膏滋正确吃法"
✅ 正确：用干净木勺取一勺、温水冲泡、空腹或饭后半小时
❌ 错误：用湿勺取用、开水直接冲、不密封保存
```

### 示例 3：症状判断

```
标题："脾虚还是湿气重？"
✅ 脾虚：饭后腹胀、大便溏稀、面色萎黄、乏力
❌ 湿气重：头昏沉、舌苔厚腻、身体沉重、皮肤出油
```
