# 单品深扒卡 — 模板

## 适用场景
信羽产品深度介绍、成分解析、适用人群分析。

## 分层 Prompt

```
[主体] 产品实拍（罐装膏/茶包/粉剂），
       放在天然亚麻布或竹席上，
       旁边搭配对应的中药材原料，展示成分透明度

[场景] 自然光台面，浅色木纹背景，
       干净简洁但不失温度，体现手作匠心感

[风格] 高端产品摄影风，微距特写，
       突出质感（膏体的光泽/茶叶的纹理/粉剂的细腻）

[构图] 竖屏 3:4，产品位于画面黄金分割点，
       周围留白，下方可放置成分小标签

[用光] 柔和的侧逆光，45度角打光，
       突出产品立体感和药材纹理，阴影柔和不生硬

[文字] 上方标题："【产品名】" 深褐色粗体
       右侧或下方小字标注核心成分：
       "【成分1】·【成分2】·【成分3】"
       底部一行小字："信羽·古法熬制"
       无其他文字或标签
```

## 产品替换指南

| 产品 | 主体描述 | 搭配药材 | 色调建议 |
|------|---------|---------|---------|
| 祛湿膏 | 深色罐装膏体，光泽感 | 茯苓+陈皮+薏米 | 暖米色+自然绿 |
| 健脾养胃膏 | 浅褐色膏体，木勺舀取 | 山药+莲子+党参 | 淡黄+暖棕 |
| 滋阴石斛膏 | 琥珀色膏体，晶莹光泽 | 石斛+麦冬+玉竹 | 金色+深绿 |
| 疏肝散结膏 | 深色膏体，药材可见 | 柴胡+夏枯草+橘核 | 淡紫+灰蓝 |
| 祛湿茶 | 散装茶叶+茶包 | 薏米+赤小豆+荷叶 | 浅绿+暖米色 |
| 养肝护肝茶 | 茶叶+小袋独立包装 | 菊花+决明子+枸杞 | 暖黄+淡绿 |

## 进阶技巧

如需展示"膏体拉丝"效果（强调古法熬制），在主体中加入：

```
木勺舀起膏体，形成晶莹的拉丝效果，
膏体呈琥珀色半透明状，粘稠有光泽
```
