# 食疗方卡 — 模板

## 适用场景
推荐食疗方、食谱、药膳。

## 分层 Prompt

```
[主体] 一碗/一盘养生食疗成品（如养生粥/汤/茶），
       摆盘精致，食材可见，冒着热气

[场景] 厨房台面或餐桌，
       自然光，有食材原料作为背景装饰

[风格] 美食摄影风，色彩鲜艳，
       小红书美食封面风格

[构图] 竖屏 3:4，俯拍或 45 度角，
       食物在画面中央，上方留文字空间

[用光] 自然顶光或侧光，突出食物的质感和色泽

[文字] 上方标题："【食疗名】" 暖色粗体
       下方列出主要食材："【食材1】+【食材2】+【食材3】"
       右下角透明标签："收藏·今晚试试"
       无其他文字
```

## 应用示例

```
标题："四神汤"
食材：山药 + 茯苓 + 莲子 + 芡实
功效标注："健脾祛湿 · 四季皆宜"
```
