# 日更 3 条 SOP（标准操作流程）

## 每天耗时预估：30-45 分钟

## 第 1 步：选题（5 分钟）

从内容日历或以下 3 个来源选 3 个选题：

1. **RAG 数据库抽取**（优先）
   ```bash
   curl -s "http://192.168.31.120:9200/search?q=养生&limit=1"
   curl -s "http://192.168.31.120:9200/products?limit=52" | python3 -c "随机选一个产品"
   ```

2. **热点日历**（换季/节气/节日）
3. **评论区/私信**（粉丝问得多的）

分配：
- 图文 1：知识科普（当天选题）
- 图文 2：产品种草（关联产品）
- 视频：数字人口播（科普/答疑）

## 第 2 步：生成图文笔记（15 分钟）

每条图文 = 封面 1 张 + 内页 3-4 张

**封面生成：**
1. 打开对应模板（templates/）
2. 替换占位符（标题、副标题、主体描述）
3. 参考配色方案（color-palettes/）调色
4. 发给 GPT Image 2 生成

**内页生成：**
每页给一个简短的 prompt，例如：
- 第 2 页：场景图（如"湿气重的舌苔对比图"）
- 第 3 页：信息图（"3 个方法"列表）
- 第 4 页：产品图或引导关注图

## 第 3 步：生成数字人视频脚本（5 分钟）

从当天的科普选题改写为口播脚本：

```
格式：
【开头】（10秒）痛点引入
"你有没有XXX的情况？很多人以为这是XXX，
其实中医看来，这可能是XXX"

【中间】（30秒）核心内容
"中医认为…… 第一…… 第二…… 第三……"

【结尾】（10秒）总结+引导
"所以如果你也有XXX，不妨试试XXX。
关注信羽，每天学点中医养生。"
```

## 第 4 步：生成数字人视频（5 分钟）

把脚本导入腾讯智影/HeyGen：
1. 选信羽的数字人形象
2. 粘贴脚本
3. 选择语音（推荐温柔女声）
4. 生成 → 下载

## 第 5 步：发布（10 分钟）

| 平台 | 发布内容 | 最佳发布时间 |
|------|---------|------------|
| 小红书 | 2 条图文 + 1 条视频 | 早上 8:00 / 中午 12:00 / 晚上 8:00 |
| 视频号 | 同步发视频 | 晚上 7:00-9:00 |
| 抖音 | 同步发视频（可选） | 中午 12:00 / 晚上 8:00 |

## 周检视（每周一次，15 分钟）

1. 上周哪条数据好？
2. 评论区有什么高频问题？
3. 调整下周选题方向
