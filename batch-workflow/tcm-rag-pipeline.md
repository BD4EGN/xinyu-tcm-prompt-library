# RAG → Prompt → 图片 全自动流水线

## 概览

利用现有基础设施实现完全自动化的图文内容生产。

## 数据流

```
RAG 数据库 (192.168.31.120:9200)
    ↓  搜索养生知识
DeepSeek 改写文案
    ↓  生成小红书文案 + 图片描述
GPT Image 2 生成图片
    ↓  生成封面 + 内页
保存到本地 → 手动检查 → 发布
```

## 手工干预节点

| 步骤 | 建议干预 |
|------|---------|
| DeepSeek 改写后 | 看一眼标题和文案，确认没问题 |
| 图片生成后 | 选一张满意的（可多生成几张选） |
| 发布前 | 最终检查排版和合规免责声明 |

其他步骤都可以自动化。
